## Run Locally

Clone the project

```bash
  git clone https://github.com/devpenzil/Bloggy.git
```

Go to the project directory

```bash
  cd Blogggy
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
